# 完了タスク管理

このページでは、完了したタスクの完了予定日と完了実績日をマトリクス形式で管理します。

## 使用方法

1. タスクリストから完了したタスクをこのページに移動
2. 以下のテーブル形式で追記してください
3. タスク番号は元のタスクリストの番号を参照してください
4. 完了予定日と完了実績日を記入してください

## 完了タスク統計

- **予定通り完了**: 1件
- **遅延完了**: 1件
- **合計**: 2件

## テンプレート（コピーして使用）

```html
<table class="task-completion-matrix">
<thead>
<tr>
<th class="task-number">番号</th>
<th>タスク内容</th>
<th class="date-cell">完了予定日</th>
<th class="date-cell">完了実績日</th>
<th>備考</th>
</tr>
</thead>
<tbody>
<tr class="on-time">
<td class="task-number">X</td>
<td>タスク内容をここに記入</td>
<td class="date-cell">YYYY-MM-DD</td>
<td class="date-cell">YYYY-MM-DD</td>
<td>備考があれば記入</td>
</tr>
</tbody>
</table>
```

**クラスの説明：**
- `task-completion-matrix`: テーブル全体に適用するクラス
- `task-number`: タスク番号のセルに適用するクラス
- `date-cell`: 日付のセルに適用するクラス
- `on-time`: 予定通り完了したタスクの行に適用するクラス（緑色のボーダー）
- `delayed`: 遅延完了したタスクの行に適用するクラス（オレンジ色のボーダー）

## 注意事項

- タスク番号は元のタスクリストの番号をそのまま使用してください
- 完了予定日と完了実績日は `YYYY-MM-DD` 形式で記入してください
- 遅延しているタスクは、CSSで自動的に左側にオレンジ色のボーダーが表示されます
- 予定通り完了したタスクは、CSSで自動的に左側に緑色のボーダーが表示されます
