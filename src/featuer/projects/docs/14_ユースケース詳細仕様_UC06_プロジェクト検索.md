# 14: UC-06 プロジェクト検索 📝

## UC-06: プロジェクト検索（将来実装）

**概要**: プロジェクト名やPME番号でプロジェクトを検索する

**アクター**: プロジェクト管理者、一般ユーザー

**前提条件**:
- 検索条件が提供されている

**基本フロー**:
1. 検索条件のバリデーション
2. プロジェクト名・PME番号で検索
3. 検索結果を返却

**例外フロー**:
- **E1**: 検索条件が無効
  - エラーメッセージ: バリデーションエラーメッセージ
  - HTTPステータス: 400 Bad Request

**後置条件**:
- 検索条件に一致するプロジェクトが返却される
